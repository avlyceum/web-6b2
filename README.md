# Nexcent - Landing Page

## Опис проекту

Nexcent - це сучасна цільова сторінка (landing page), розроблена для представлення послуг та можливостей бізнесу. Проект демонструє професійний підхід до веб-дизайну з використанням HTML, CSS та сучасних практик.

## Особливості

- 🎨 **Сучасний дизайн** - використання актуальних шрифтів Google Fonts (Inter)
- 📱 **Адаптивна верстка** - оптимізована для різних розмірів екранів
- 🎯 **Чистий код** - структурований та легко читаємий HTML та CSS
- 🚀 **Швидке завантаження** - мінімалістичний підхід без зайвих залежностей
- 🎨 **Гарна кольорова схема** - професійна палітра з зеленим акцентним кольором

## Структура проекту

```
.
├── index.html      # Основна сторінка
├── style.css       # Стилі проекту
├── logo.svg        # Логотип бренду
├── Illustration.png # Ілюстрація для героїки
└── README.md       # Цей файл
```

## Розділи сторінки

- **Навігація** - меню з посиланнями на основні розділи
- **Герой секція** - привітальний екран з основним повідомленням та кнопкою реєстрації
- **Сервіси** - описання послуг
- **Можливості** - основні переваги
- **Продукти** - перелік пропонованих продуктів
- **Відгуки** - testimоніали від клієнтів
- **FAQ** - відповіді на часті питання

## Технологічний стек

- HTML5
- CSS3 (з CSS змінними)
- Google Fonts (Inter)
